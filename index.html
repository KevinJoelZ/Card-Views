HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Es utilizado para estructurar el contenido en la web.

CSS

CSS (Cascading Style Sheets) es el lenguaje que usamos para estilizar un documento HTML. CSS describe cómo los elementos HTML deben ser mostrados en pantalla.

JavaScript

JavaScript es un lenguaje de programación que permite implementar funciones complejas en páginas web. Es la tercera capa del estándar de tecnologías web.
